# Ariadne API

Simple Node.js API server for Ariadne project.

## Scripts

- `pnpm dev` - Start the development server (port 3004)
- `pnpm start` - Start the production server (port 3004)

## Endpoints

- `GET /` - Welcome message
- `GET /health` - Health check endpoint

## Environment Variables

- `PORT` - Server port (default: 3004)
