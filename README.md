## Run locally
1) Install serve (once):
   npm install -g serve

2) Start server:
   cd C:\Users\Andy\andysd-suite
   serve . -l 3000

Open:
- http://localhost:3000
- http://localhost:3000/work/gigos/
- http://localhost:3000/work/flippin/
